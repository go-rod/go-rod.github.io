# Rod

Rod é um driver de alto nível para o [protocolo DevTools](https://chromedevtools.github.io/devtools-protocol). Ele é amplamente usado para automação e extração da web. Rod pode automatizar a maioria das coisas no navegador que podem ser feitas manualmente:

- Capturar capturas de página
- Extrair dados em páginas renderizadas no lado do cliente
- Preencher formulários
- Teste End-to-end (e2e)
- Interceptar requisições
- E mais
