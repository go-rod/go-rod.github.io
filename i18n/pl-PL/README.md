# Pręt

Rod jest sterownikiem wysokiego poziomu dla [protokołu DevTools](https://chromedevtools.github.io/devtools-protocol). Jest powszechnie używany do automatyzacji i złomowania stron internetowych. Rod może zautomatyzować większość rzeczy w przeglądarce, które można wykonać ręcznie:

- Przechwyć zrzuty ekranu strony
- Pokrywa renderowane strony po stronie klienta
- Formularze autouzupełniania
- Badanie na końcu
- Żądania Hijack
- I więcej

Aby przyczynić się do dokumentacji, [sprawdź tutaj](contribute-doc.md).
